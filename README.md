# Daily Planner 🗓️

A simple web app to manage you daily tasks.

## 🧠 Features
- Add new tasks
- Complete tasks
- Delete tasks
- Search tasks
- Save tasks in localStorage


## How to Use
1. Open `index.html` in your browser
2. Type a task and press "Add Task"
3. Check the box when done
4. Use the search box to find tasks
